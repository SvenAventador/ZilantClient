# ZilantClinet

This project is intended for a term paper and then a thesis. This is the client part
To start the server, you need to create an environment variables file `.env` to store your server's link, for example, `protokol://host:port`. Next, you need to write the command `npm install` to install all the dependencies in your project, and at the end write the command "npm start" so that the site starts working.
